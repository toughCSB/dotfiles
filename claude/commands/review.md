## 검토 항목

### 1. 정확성
- 로직 오류, 엣지케이스 누락
- 타입 불일치, null/undefined 처리

### 2. 보안
- SQL Injection, XSS, 인증 우회 가능성
- 비밀값이 하드코딩되어 있는지

### 3. 성능
- 불필요한 루프, N+1 쿼리
- 메모리 누수 가능성

### 4. 가독성
- 함수/변수명이 의도를 드러내는지
- 중복 코드, 불필요한 복잡성

### 5. CLAUDE.md 준수
- 요청 범위 이상의 변경은 없는지
- 기존 코드 스타일을 따르는지
