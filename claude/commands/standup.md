오늘의 스탠드업 리포트를 생성합니다. 최근 git 활동과 TODO를 기반으로 작성합니다.

## 수집 정보

1. **어제 한 일** - `git log --since="yesterday" --oneline --author="$(git config user.email)"` 로 커밋 확인
2. **오늘 할 일** - 현재 브랜치의 열린 작업 파악 (최근 커밋 패턴, TODO 주석)
3. **블로커** - 미완료 또는 실패한 작업 있는지 확인

## 출력 형식

```
📅 스탠드업 - YYYY-MM-DD

✅ 어제 한 일
- ...

🎯 오늘 할 일
- ...

⚠️ 블로커
- (없음) / ...
```

## 팁
- GWS 캘린더 확인이 필요하면 `gws-calendar-agenda` 스킬 활용
- 특정 프로젝트 범위를 지정하려면 인수로 전달: `/standup 프로젝트명`
